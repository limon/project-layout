## File layout for node module development

## Main components:

* gulp
* mocha/chai
* babel

## Get started:

	npm install
	gulp                # default = watch

## To update versions of package.json dependencies:

	$ npm outdated
    # or
	$ npm install -g npm-check-updates
	$ npm-check-updates
    # update package.json with new versions if you agree
	$ npm-check-updates -u
